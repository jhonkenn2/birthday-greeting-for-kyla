# Interactive Birthday Greeting Cake with Card

## Project Overview

This project is an interactive birthday greeting webpage that combines a digital birthday cake animation with a personalized greeting card experience. The goal is to create a fun, visually appealing, and interactive birthday surprise that users can open, interact with, and enjoy directly in the browser.

---

# Project Title

**Interactive Birthday Greeting Cake with Card**

---

# Objectives

* Create an interactive birthday greeting experience
* Display an animated birthday cake with candles
* Add a greeting card that can open and reveal a custom message
* Provide smooth animations and responsive design
* Make the experience enjoyable on both desktop and mobile devices

---

# Main Features

## 1. Welcome Screen

* Beautiful birthday-themed landing section
* Start button to begin the experience
* Animated background effects such as confetti, balloons, or floating particles

## 2. Interactive Birthday Cake

* Multi-layer birthday cake design
* Animated candles with glowing flame effects
* Blow candle interaction using button click or microphone feature (optional)
* Cake animation when candles are blown out

## 3. Birthday Greeting Card

* Foldable/openable greeting card animation
* Personalized birthday message inside the card
* Smooth card opening transition
* Decorative birthday design elements

## 4. Music and Sound Effects

* Optional happy birthday background music
* Candle blow sound effects
* Card opening sound effects
* Music control button for mute/unmute

## 5. Celebration Effects

* Confetti explosion after interaction
* Floating balloons or sparkles
* Animated text greetings
* Fireworks or glowing effects

# User Flow

1. User opens the webpage
2. Welcome screen appears
3. User clicks the start button
4. Birthday cake animation is shown
5. User interacts with the candles
6. Greeting card opens automatically
7. Birthday message is displayed
8. Celebration animations appear
